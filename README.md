# Alerto24 - Emergency Service Platform
## Backend (Django/GraphQL)
- Django 4.2 with PostgreSQL + PostGIS
- GraphQL API with JWT authentication
- Apps: users, emergencies, providers, notifications
- Real-time WebSocket support
- Legacy API integration layer

## Frontend (React Admin)
- React 18 + TypeScript + Vite
- Material-UI components
- Apollo GraphQL client
- Dashboard with real-time updates

## Mobile (Flutter)
- Flutter 3.13 for iOS & Android
- GraphQL integration
- Real-time location tracking
- Emergency creation flow

## Infrastructure
- Docker Compose for local development
- PostgreSQL + Redis services
- Kubernetes manifests for production
- Development scripts

## Key Features
- Emergency reporting and dispatch
- Real-time provider tracking
- Multi-role authentication (citizen/provider/admin)
- Legacy 911/112 system integration
- Push notifications and SMS alerts
